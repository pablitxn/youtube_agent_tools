# Getting Started

Welcome to YouTube RAG Server! This section will help you get up and running quickly.

## Overview

YouTube RAG Server enables you to:

1. **Ingest** YouTube videos - download, transcribe, and extract frames
2. **Index** content - create searchable embeddings across all modalities
3. **Query** semantically - ask questions and get answers with precise citations
4. **Retrieve** sources - get video clips, frames, and audio at exact timestamps

## Prerequisites

Before you begin, ensure you have:

- **Python 3.11+** installed
- **Docker** and **Docker Compose** for local infrastructure
- **FFmpeg** installed on your system
- **OpenAI API key** for transcription and embeddings
